# 快速入门

您无需对代码或测试设置进行任何更改。如果您使用 Test Replay 将测试运行记录到 Cypress Cloud，就可以开始使用 Cypress 无障碍测试功能。
您将通过销售团队的个性化支持开始免费试用，利用现有测试数据立即查看所有 Cypress 项目的完整无障碍报告。
之后，您可以集成 CI 来设置自己的结果处理标准。

## 1. 申请试用

开始免费试用 Cypress 无障碍测试功能。您将获得个性化演示，使用现有测试数据查看所有 Cypress 项目中存在的测试盲点。

## 2. 记录测试运行

Cypress 无障碍测试使用 [Test Replay](/cloud/features/test-replay) 数据生成报告，需要 Cypress v13+ 版本。开始使用无需额外配置或插桩。

使用 Test Replay [记录测试运行](/cloud/get-started/setup) 到 Cypress Cloud 即可开始使用无障碍测试功能。

## 3. 查看无障碍测试结果

记录测试运行后，您可以在 Cypress Cloud 的 **无障碍测试** 选项卡中查看结果。

## 4. 自定义结果

Cypress 无障碍测试提供灵活的配置选项来自定义报告。您可以配置：

- [元素过滤器](/accessibility/configuration/elementfilters)：指定应从无障碍扫描中排除的元素选择器
- [视图过滤器](/accessibility/configuration/viewfilters)：指定应从无障碍扫描中排除的 URL 模式。排除 URL 也会排除指向该 URL 的所有链接
- [视图](/accessibility/configuration/views)：指定代表视图的 URL 模式

要添加或修改项目配置，请导航至项目设置中的 **应用质量** 选项卡。

## 5. 查看详细指南

Cypress 无障碍测试支持许多可能不太明显的不同无障碍自动化工作流，例如大型内容管理网站的生产环境监控。还提供了 API 用于编程访问每次运行的结果，以及 Branch Review 中的报告比较功能，可以仅审查特定运行中发现的新违规项。这些指南详细介绍了 Cypress 无障碍测试支持的许多场景和用例，帮助您了解如何在自己的团队中应用这些结果。
