# 监控变更

随时间监控UI覆盖率分数的变化能确保在代码合并前识别并解决回归问题。Cypress UI覆盖率提供工具来追踪这些变更，使您能在开发工作流中实现主动的质量保障。

## 使用Results API实现自动化监控

UI覆盖率的[Results API](/ui-coverage/results-api)允许您以编程方式获取UI覆盖率数据，以便集成到CI/CD流水线中。

## 通过Branch Review深入分析变更

[UI覆盖率的Branch Review](/ui-coverage/guides/compare-reports)功能允许您详细比较任意两次运行结果，审查新增的未测试元素和链接，以及应用中所有新增的交互元素。

## 通过跨项目分析发现长期趋势

在Cypress Cloud的[企业报告](/cloud/features/analytics/enterprise-reporting#UI-Coverage)区域，您可以查看所有项目和运行的高层趋势，下载报告或通过API获取详细信息。
