# elementFilters

<ElementFilters />

### 排除禁用元素

UI 覆盖率默认会高亮显示禁用元素，以揭示应用中可能存在的未测试路径（以防这些元素后续变为启用状态）。如果需要，你可以使用 `elementFilters` 配置将它们从报告中排除。

#### 配置

```json
{
  "elementFilters": [
    {
      "selector": "[disabled]",
      "include": false
    }
  ]
}
```

#### HTML

```xml
<body>
  <button data-cy='cancel'>取消</button>
  <button disabled data-cy='submit'>提交</button>
</body>
```

#### UI 中显示的元素

```
[data-cy='cancel']
```
