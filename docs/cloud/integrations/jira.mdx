# Jira集成

:::info

##### <Icon name="question-circle" color="#4BBFD2" /> 你将学到

- 如何安装Cypress Jira集成
- 如何从Cypress Cloud为测试用例创建Jira问题

:::

[Cypress Cloud](https://on.cypress.io/cloud)可以与您的Jira工作流集成，实现：

- 直接从Cypress Cloud为测试用例创建一个或多个Jira问题
- 查看跨测试运行的测试用例关联或创建的Jira问题历史记录

## 安装Jira集成

:::tip

<Icon name="star" /> **Cypress Cloud高级功能**

**Jira集成**适用于拥有[团队版Cypress Cloud计划](https://cypress.io/pricing)的用户。

:::

1. 在Cypress Cloud中访问**集成 → Jira**，点击**安装Jira**
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-install.png"
     alt="Cypress Cloud集成"
   />
1. 在[Atlassian Marketplace Cypress for Jira页面](https://marketplace.atlassian.com/apps/1224341/cypress-for-jira?hosting=cloud&tab=overview)点击**立即获取**
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-atlassian-get-it-now.png"
     alt="Atlassian Marketplace Cypress for Jira"
   />
1. 登录Jira，选择安装站点并点击**安装应用**
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-atlassian-choose-site.png"
     alt="Atlassian Marketplace Cypress for Jira选择站点"
   />
1. 在"添加到Jira"界面，点击**立即获取**确认安装
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-atlassian-confirm-install.png"
     alt="Atlassian Marketplace Cypress for Jira安装确认"
   />
1. 安装完成后，点击通知中的**开始使用**链接
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-atlassian-success.png"
     alt="Atlassian Marketplace Cypress for Jira安装成功"
   />
1. 在下一页，点击**点击完成安装**按钮，重定向到[Cypress Cloud](https://cloud.cypress.io/)并选择要集成的Cypress组织
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-atlassian-finish-install-step.png"
     alt="Atlassian Marketplace Cypress for Jira完成安装步骤"
   />
1. 重定向到[Cypress Cloud](https://cloud.cypress.io/)后，选择要与[Atlassian Marketplace Cypress for Jira](https://marketplace.atlassian.com/apps/1224341/cypress-for-jira?hosting=cloud&tab=overview)集成的组织
   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-choose-cypress-org.png"
     alt="Cypress Cloud选择Jira集成组织"
   />
1. Jira集成完成。Jira集成页面会显示允许的项目列表

   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-completed.png"
     alt="Cypress Cloud Jira集成"
   />

:::caution

<Icon name="exclamation-triangle" /> 仅支持一个活动安装

:::

## 为测试用例创建Jira问题

:::info

<strong>何时为测试创建问题？</strong>

可以为通过或失败的测试创建Jira问题。最常见的使用场景是为失败的测试创建Jira问题，以便更好地跟踪和优先处理由Jira工作流管理的项目中的流水线故障修复。

:::

让我们通过为失败的测试创建Jira问题：

1. 在测试运行的"测试结果"视图中点击失败的测试，打开其结果面板

2. 点击结果面板中的"创建问题"按钮，将显示为测试创建Jira问题的表单

   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-create-issue.png"
     alt="创建Jira问题按钮"
   />

3. 通过选择Jira项目、问题类型、分配人和附加字段，完成并提交Jira问题创建表单

   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-modal.png"
     alt="创建Jira问题模态框"
   />

4. 问题创建后，测试结果面板中会出现问题的日志和引用

   <DocsImage
     src="/img/cloud/integrations/jira/dashboard-jira-integration-inline-issue.png"
     alt="Cypress失败上下文中的Jira问题"
   />

:::info

Jira中创建的问题将包含返回Cypress Cloud的关联测试链接

<DocsImage
  src="/img/cloud/integrations/jira/dashboard-jira-integration-jira-issue.png"
  alt="Cypress失败的Jira问题"
/>

<DocsImage

  alt="Cypress失败的Jira问题"
/>

:::

## 卸载Jira集成

您可以通过访问组织的**集成 → Jira**页面移除此集成。这将禁用从测试失败创建Jira问题的功能。
