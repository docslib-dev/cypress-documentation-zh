# 企业级报告

:::info

##### <Icon name="question-circle" color="#4BBFD2" /> 您将了解

- 如何在Cypress Cloud中访问和使用企业级报告
- 如何解读企业级报告提供的数据和洞察
- 如何从企业级报告下载数据
- 如何通过Cypress Cloud API以编程方式获取测试数据

:::

Cypress Cloud提供企业级报告功能，让您能够发现趋势和异常、跟踪使用情况，并管理组织内所有项目的整体测试计划。该功能包含可交互的可视化卡片，突出显示使用情况、测试指标、测试类型、不稳定测试、构建版本等信息。Cloud中展示的所有图表原始数据都可以多种格式下载。如果您更倾向于以编程方式获取测试数据用于自己的分析平台，请查看我们的[数据提取API](/cloud/integrations/data-extract-api)。

:::tip

<Icon name="star" /> **Cypress Cloud高级功能**

**企业级报告**仅适用于拥有
[企业版Cypress Cloud计划](https://cypress.io/pricing)的用户。
:::

## 全面洞察与数据访问

如果您的Cypress计划包含此功能，可通过左侧菜单中的**企业级报告**链接访问。

您的洞察数据按标签页分组显示，包括**总体使用情况**、**测试结果**、**测试设置**、**测试执行**、**不稳定测试**和**原始数据**。页面首次加载时，会显示您组织中所有项目当前月及前两个月的数据。但您可以通过页面上的筛选器自定义日期范围和项目。

**需要注意几点：**

- 企业级报告中显示的数据并非实时数据。数据更新截至当天UTC时间午夜，即数据会更新到"昨天结束"。
- 历史数据的可用性取决于您在Cypress Cloud中的数据保留期限。

以下各节将描述各标签页上可用的信息类型和洞察内容。

### 总体使用情况

- 您的组织在所有项目中使用了多少Cypress测试？
- 这些测试集中在哪些地方？
- 测试使用情况随时间发生了怎样的变化？

这些问题可以通过企业级报告**总体使用情况**标签页上的卡片得到解答。如需深入特定时间段，可使用`开始日期`和`结束日期`筛选器。`项目`筛选器可用于按一个或多个项目筛选数据。`分支`筛选器可用于选择项目中存在的一个或多个分支。此标签页上的数据与Cypress Cloud[计为测试结果的内容](/cloud/faq#What-counts-as-a-test-result)相关。

### 测试设置

企业级报告的**测试设置**标签页包含显示您的运行如何配置和维护的指标。

- 您的端到端测试与[组件测试](/app/component-testing/get-started)的比例是多少？
- 您的测试套件规模在组织中随时间发生了怎样的变化？
- 哪些项目配置并使用了Cypress智能编排功能？
- 您用于运行的Cypress版本是什么？

这些问题将通过此标签页上展示的数据得到解答。

### 测试执行

企业级报告的**测试执行**标签页帮助您了解Cypress运行的环境和细节。

- 测试了哪些浏览器？
- 用于测试的浏览器在组织中是否一致，或在项目中是否随时间变化？
- 您的Cypress运行的测试执行时间是否有变化？
- 您组织中测试的平均失败或错误率是多少，单个项目与这些平均值相比如何？

这些问题将通过此标签页上展示的数据得到解答。

### 不稳定测试

企业级报告的**不稳定测试**标签页允许您查看整个组织或特定项目的不稳定测试发生情况和比率。这可以是基于开始和结束日期筛选器的单一快照，也可以随时间查看。不稳定_率_很有帮助，因为它可以让您看到项目中不稳定运行的百分比。这使得无论项目中的运行量如何，都可以轻松比较项目之间的不稳定测试。

### 从Cloud下载数据

企业级报告中显示的所有图表都允许您下载用于渲染图表可视化的原始数据。如果您想手动下载原始数据进行进一步审查，只需将鼠标悬停在图表上，从图表右上角的三点菜单中选择`下载结果`，将显示一个菜单，允许您以以下格式下载数据：

- .csv
- .xlsx
- .json
- .png

选择您想要下载的格式后，文件将立即由您的浏览器下载。

<DocsImage
  src="/img/cloud/features/analytics/enterprise-reporting-download.png"
  alt="Cloud企业级报告下载数据"
/>

### 通过API获取测试数据

Cypress Cloud的[数据提取API](/cloud/integrations/data-extract-api)提供对Cloud企业级报告页面中显示的测试数据的完全访问。**原始数据**标签页上提供了粒度数据的示例，以便您在决定与Cypress Cloud API集成之前可以审查数据。要下载，只需将鼠标悬停在图表上，选择`下载结果`，并选择您数据的下载格式。当您准备好与API集成时，可以访问[数据提取API](/cloud/integrations/data-extract-api)页面，查看端点列表、返回的数据以及集成说明。
