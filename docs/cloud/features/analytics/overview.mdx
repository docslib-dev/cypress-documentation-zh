# 分析类型

Cypress Cloud 帮助您从多个角度理解测试套件的各个方面。
通过查看测试用例统计、常见错误、测试运行时长、不稳定率等指标，您可以监控和管理应用程序质量。

- [项目级分析](/cloud/features/analytics/project-analytics): 
  为组织中的每个项目提供分析功能，便于您了解单个项目的趋势和详细信息。
- [企业级报告](/cloud/features/analytics/enterprise-reporting): 
  提供跨组织的分析功能，帮助您发现项目间的模式或异常情况，跟踪测试使用情况，确保一致性。此级别的分析使您能够管理整个组织的测试计划。
- [数据提取API](/cloud/integrations/data-extract-api): 
  可以从 Cypress Cloud 以多种格式提取聚合数据和原始测试数据。获取测试数据用于您自己的分析系统，或自动化其他可根据您选择的条件触发的工作流程。
