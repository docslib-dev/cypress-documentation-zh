# 项目分析与洞察

Cypress Cloud 提供分析功能，可洞察随时间变化的运行次数、运行时长以及测试套件规模的可视化数据。

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-overview.png"
  alt="Cloud Analytics Screenshot"
/>

## 运行状态

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-runs-over-time.png"
  alt="Cloud Analytics Runs Over Time Screenshot"
/>

该报告显示您的组织记录到 Cypress Cloud 的运行次数，按运行的最终状态分类。每次运行代表对该项目执行的一次 `cypress run --record` 调用，无论是在 CI 还是本地机器上。

### 筛选条件

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-runs-over-time-filters.png"
  alt="Cloud Analytics Runs Over Time Filters Screenshot"
/>

可按以下条件筛选结果：

- 分支
- 时间范围
- 时间间隔（每小时、每日、每周、每月、每季度）

### 结果

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-runs-over-time-graph.png"
  alt="Cloud Analytics Runs Over Time Graph Screenshot"
/>

随时间变化的总运行次数会显示通过、失败、运行中、超时和错误的测试数量，根据所选筛选条件而定。

结果可下载为逗号分隔值（CSV）文件以供进一步分析。可通过筛选条件右侧的下载图标完成此操作。

### 关键绩效指标

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-runs-over-time-kpi.png"
  alt="Cloud Analytics Runs Over Time KPI Screenshot"
/>

总运行次数、日均运行次数、通过运行次数和失败运行次数是根据所选筛选条件计算得出的。

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-runs-over-time-table.png"
  alt="Cloud Analytics Runs Over Time Table Screenshot"
/>

显示按时间范围筛选条件分组的日期结果表格，包含通过、失败、运行中、超时和错误的列。

- 标签
- 时间范围
- 时间间隔（每小时、每日、每周、每月、每季度）

### 结果

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-run-duration-graph.png"
  alt="Cloud Analytics Run Duration Graph Screenshot"
/>

随时间变化的平均运行时长根据所选筛选条件显示。

结果可下载为逗号分隔值（CSV）文件以供进一步分析。可通过筛选条件右侧的下载图标完成此操作。

### 关键绩效指标

  alt="Cloud Analytics Run Duration KPI Screenshot"

<DocsImage
  src="/img/cloud/features/analytics/dashboard-analytics-run-duration-table.png"
  alt="Cloud Analytics Run Duration Table Screenshot"
/>

显示按时间范围筛选条件分组的日期结果表格，包含平均运行时间、并发数和并行化节省时间的列。
