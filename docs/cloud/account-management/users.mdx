# 用户管理

:::info

##### <Icon name="question-circle" color="#4BBFD2" /> 学习内容

- 如何邀请用户加入Cypress Cloud
- 如何管理用户角色和更新
- 如何处理用户请求

:::

用户是指任何登录Cypress Cloud的人员。

## 邀请用户

组织[所有者或管理员](#用户角色)可以邀请用户加入[Cypress Cloud](https://on.cypress.io/cloud)。被邀请的用户将能看到他们拥有[团队权限](/cloud/account-management/teams#为团队分配项目)的项目和测试。

_注意：只有所有者可以授予其他用户"所有者"权限。_

### 邀请用户加入组织：

1. 选择要邀请用户的[组织](/cloud/account-management/organizations)
2. 点击**用户与团队**，然后点击**邀请用户**
3. 添加一个或多个邮箱地址（每个地址输入后按**Enter**键），并选择他们的[角色](#用户角色)
4. 点击**邀请用户**
5. 用户将收到包含接受邀请链接的邮件

如果您希望自行发送邀请信息并附带加入组织的链接，可以点击**复制邀请链接**，所有接受此邀请的用户将被默认授予**成员**角色。

<DocsImage
  src="/img/cloud/users/invite-user-dialog.jpg"
  alt="邀请用户对话框"
/>

### 通过邀请注册：

收到邮件后，点击**"接受邀请"**跳转至Cypress Cloud。您可以选择以下认证方式：

- **基础认证**（邮箱和密码）
- **社交账号**（Google或GitHub）
- **单点登录(SSO)**

#### **基础认证注册**

1. 点击**"没有账号？注册"**
2. 点击**"通过邮箱注册"**
3. 输入**邮箱地址**并设置**密码**
4. 点击**"注册"**

#### **社交账号注册**

1. 点击**"没有账号？注册"**
2. 点击**"通过Google注册"**或**"通过GitHub注册"**
3. 根据您的个人设置，可能需要通过社交账号提供商进行验证

#### **单点登录(SSO)注册**

请参考我们的[企业SSO指南](/cloud/account-management/enterprise-sso)。

#### 账户锁定或忘记密码

参见[无法访问Cypress Cloud账户怎么办](/cloud/faq#无法访问Cypress-Cloud账户怎么办)。

## 用户角色

用户可被分配不同角色，这些角色影响其对[Cypress Cloud](https://on.cypress.io/cloud)功能的访问权限。

- **查看者**：可查看项目和运行记录，但不能进行任何修改
- **成员**：可查看项目、运行记录、录制密钥，并可调整应用质量配置
- **团队管理员**：可管理项目成员列表（需本身是项目成员），并可编辑项目配置
- **组织管理员**：可管理组织内所有用户，并可编辑组织和项目配置
- **组织所有者**：可管理组织和项目的所有方面

## 用户请求

Cypress App用户可"申请"访问特定组织。如果您团队的开发人员已拥有Cypress和项目源代码的访问权限，他们可以直接从Cypress App申请加入您的组织。这意味着您无需提前邀请团队成员，他们可以主动申请，由您决定是否批准。

_注意：如果您的Cloud组织配置了SSO，此选项不可用。_

## 用户更新

如需更改用户邮箱，**所有者**或**管理员**可以邀请新用户/邮箱地址并删除旧用户/邮箱地址。

更新账单邮箱地址可通过[Cypress Cloud](https://on.cypress.io/cloud)中的**账单与使用情况**页面，或直接联系[billing@cypress.io](mailto:billing@cypress.io)。
