# 组织管理

:::info

##### <Icon name="question-circle" color="#4BBFD2" /> 您将学习到

- 如何创建和管理组织
- 如何查找您的组织ID
- 如何为您的组织集成源代码管理提供商、问题管理和团队沟通工具

:::

组织用于分组和管理项目。在这里您可以创建项目、管理用户、访问账单和使用信息、利用第三方集成以及应用组织级别的设置。

## 组织ID

组织ID是一个采用[UUID](https://tools.ietf.org/html/rfc4122)格式的唯一标识符，可在Cypress Cloud的"组织设置"部分找到。支持、成功和销售团队经常使用此ID来引用组织。

要查找您的组织ID，请转到"组织设置"，然后查看组织ID。点击剪贴板图标可复制ID，方便与Cypress团队共享。

/>

## 管理组织

### 创建组织

您可以通过打开组织切换器并点击**<Icon name="plus" /> 创建新组织**，在[Cypress Cloud](https://on.cypress.io/cloud)中创建组织。

<DocsImage
  src="/img/cloud/organizations/create-org.jpg"
  alt="创建新组织"
/>

### 管理项目

创建组织后，系统会为您创建一个新的"默认项目"。您可以在此调整[项目设置](/cloud/account-management/projects#Project-settings)以满足需求。

### 管理用户和团队

组织所有者或管理员角色可以[邀请Cypress Cloud用户](/cloud/account-management/users)加入其组织。被邀请的用户可以[分配到团队](/cloud/account-management/teams)。团队可帮助您控制哪些用户有权访问特定项目和测试。

### 管理Cloud AI

[Cloud AI](/cloud/faq#What-is-Cloud-AI)默认对所有Cloud计划的用户启用。

组织管理员和所有者可以从组织设置中为整个组织启用或禁用AI功能。

<DocsImage
  src="/img/faq/questions/cypress-ai-org-setting.png"
  alt="显示Cypress AI组织设置的图片"
/>

### 删除组织

只要组织中没有任何项目，您就可以删除您拥有的组织。在删除组织之前，您必须先[删除](/cloud/account-management/projects#Delete-Project)或[转移所有权](/cloud/account-management/projects#Transfer-ownership)将项目转移到另一个组织。

<DocsImage
  src="/img/cloud/organizations/remove-organization-dialog.jpg"
  alt="删除组织"
  width="80%"
/>
