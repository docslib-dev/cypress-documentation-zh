## 开源方案

为支持社区，我们提供开源软件(OSS)方案，包含5个免费用户、每月10万次测试、负载均衡并行测试以及众多集成功能！

申请资格要求您组织内的所有项目必须满足以下条件：

- 属于非商业实体
- 在Cypress Cloud中设置为[公开项目](/cloud/account-management/projects#Change-project-access)
- 项目源代码需公开且采用[OSI批准的开源协议](https://opensource.org/licenses)
- 至少记录500次[测试](/cloud/faq#What-counts-as-a-test-result)，该数量包含在[入门方案](https://www.cypress.io/pricing)中

### 为组织申请OSS方案

:::info

<Icon name="exclamation-triangle" /> 个人组织无法使用OSS方案。

:::

1. [登录](https://on.cypress.io/dashboard)Cypress仪表盘，或[创建账户](https://on.cypress.io/dashboard)（新用户）

2. 进入[组织页面](https://on.cypress.io/dashboard/organizations)选择要关联OSS方案的组织。若无组织，可点击**+创建新组织**按钮新建

   <DocsImage
     src="/img/cloud/organizations/oss-plan-2-select-org.jpg"
     alt="选择或添加组织"
     width="80%"
   />

3. 进入**计费与使用**页面，点击**升级**，然后点击页面底部的**申请开源方案**链接（注意：必须将所有项目访问权限设置为**公开**才能显示该链接）

   <DocsImage
     src="/img/cloud/organizations/oss-plan-3-billing.jpg"
     alt="升级方案"
   />

4. 填写并提交OSS方案申请表

   <DocsImage
     src="/img/cloud/organizations/oss-plan-5-billing.jpg"
     alt="OSS方案申请表"
   />

5. 您将收到确认邮件。Cypress团队审核通过后，将为您的组织开通OSS方案订阅

如有任何关于OSS方案的疑问，欢迎随时[联系我们](mailto:hello@cypress.io)。
