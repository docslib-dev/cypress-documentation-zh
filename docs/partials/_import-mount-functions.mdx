不同框架渲染组件的方式不同，因此我们提供了针对特定框架的 `mount()` 函数，可以通过以下方式导入：

<Tabs>
<TabItem value='React'>

:::info

<strong>React 用户须知</strong>

从 [cypress/react](https://github.com/cypress-io/cypress/tree/develop/npm/react) 模块导出的 `mount()` 命令支持标准的 JSX 语法来挂载组件。

:::

```js
import { mount } from 'cypress/react'
```

</TabItem>

<TabItem value='Vue'>

:::info

<strong>Vue 用户须知</strong>

从 [cypress/vue](https://github.com/cypress-io/cypress/tree/develop/npm/vue) 库导出的 `mount()` 命令内部使用了 [Vue Test Utils](https://vue-test-utils.vuejs.org/)，但与在 node 的虚拟浏览器中挂载组件不同，它会在实际浏览器中挂载组件。

:::

```js
import { mount } from 'cypress/vue'
```

</TabItem>
<TabItem value='Angular'>

:::info

<strong>Angular 用户须知</strong>

从 [cypress/angular](https://github.com/cypress-io/cypress/tree/develop/npm/angular) 库导出的 `mount()` 命令内部使用了 [Angular TestBed](https://angular.io/api/core/testing/TestBed)，但与在 node 的虚拟浏览器中挂载组件不同，它会在实际浏览器中挂载组件。

:::

```ts
import { mount } from 'cypress/angular'
```

</TabItem>
</Tabs>
