# 如何使用Cypress插件

:::info

##### <Icon name="question-circle" color="#4BBFD2" /> 你将学到

- 如何安装插件
- 如何使用插件

:::

Cypress维护了一个由我们和社区创建的精选插件列表。
来自我们[官方列表](/app/plugins/plugins-list)的插件都是npm模块。这使得它们可以
独立进行版本控制和更新，而无需更新Cypress本身。

你可以使用npm安装任何已发布的插件：

<Tabs>
  <TabItem value="npm">

```shell
npm install <插件名称> --save-dev
```

  </TabItem>

  <TabItem value="yarn">

```shell
yarn add <插件名称> --dev
```

  </TabItem>

  <TabItem value="pnpm">

```shell
pnpm add --save-dev <插件名称>
```

  </TabItem>
</Tabs>

## 使用插件

将你的插件添加到[Cypress配置](/app/references/configuration)中的
[`setupNodeEvents`](/app/references/configuration#setupNodeEvents)
函数。请遵循插件文档提供的任何额外说明。
以下是一个示例：

:::cypress-config-plugin-example

```ts
// 绑定我们关心的事件
on('<事件>', (arg1, arg2) => {
  // 插件相关操作在这里
})
```

:::

有关编写插件的信息，请查看我们的
[Node Events概述](/api/node-events/overview)。
