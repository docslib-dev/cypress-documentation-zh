## 框架配置

Cypress组件测试开箱即支持[Vite](https://vitejs.dev/)和自定义[Webpack](https://webpack.js.org/)配置。Cypress在设置过程中会自动检测这些框架并进行正确配置。以下示例供参考。

### 使用Vite的Svelte

Cypress组件测试支持使用Vite 4+作为打包工具的Svelte应用。

#### Vite配置

:::cypress-config-example

```ts
{
  component: {
    devServer: {
      framework: 'svelte',
      bundler: 'vite',
    },
  },
}
```

:::

#### Svelte Vite示例应用

- [Svelte 5 Vite 6 带Typescript](https://github.com/cypress-io/cypress-component-testing-apps/tree/main/svelte-vite-ts)

### 使用Webpack的Svelte

Cypress组件测试支持使用Webpack 4+作为打包工具的Vue应用。

#### Webpack配置

:::cypress-config-example

```ts
import webpackConfig from './webpack.config'
```

```ts
{
  component: {
    devServer: {
      framework: 'svelte',
      bundler: 'webpack',
      // 可选传入webpack配置
      webpackConfig,
    },
  },
}
```

:::

如果不提供配置，Cypress会尝试推断你的webpack配置。如果Cypress无法推断或你需要修改配置，可以通过`webpackConfig`选项手动传入。

#### Svelte Webpack示例应用

- [Svelte 5 Webpack 5 带Typescript](https://github.com/cypress-io/cypress-component-testing-apps/tree/main/svelte-webpack-ts)
