## 框架配置

Cypress组件测试开箱即用支持[Vite](https://vitejs.dev/)和自定义[Webpack](https://webpack.js.org/)配置。Cypress在设置过程中会自动检测这些框架并进行正确配置。以下示例仅供参考。

### Vue与Vite

Cypress组件测试支持使用Vite 4+作为打包工具的Vue应用。

#### Vite配置

:::cypress-config-example

```ts
{
  component: {
    devServer: {
      framework: 'vue',
      bundler: 'vite',
    },
  },
}
```

:::

#### Vue Vite示例应用

- [Vue 3 Vite与TypeScript](https://github.com/cypress-io/cypress-component-testing-apps/tree/main/vue3-vite-ts)

### Vue与Webpack

Cypress组件测试支持使用Webpack 4+作为打包工具的Vue应用。

#### Webpack配置

:::cypress-config-example

```ts
import webpackConfig from './webpack.config'
```

```ts
{
  component: {
    devServer: {
      framework: 'vue',
      bundler: 'webpack',
      // 可选传入webpack配置
      webpackConfig,
      webpackConfig: async () => {
        // ...处理逻辑...
        const modifiedConfig = await injectCustomConfig(baseConfig)
        return modifiedConfig
      },
    },
  },
}
```

:::

如果不提供配置，Cypress会尝试推断您的webpack配置。如果Cypress无法推断或您希望对配置进行修改，可以通过`webpackConfig`选项手动传入。

#### Vue Webpack示例应用

- [Vue 3 Webpack 5与TypeScript](https://github.com/cypress-io/cypress-component-testing-apps/tree/main/vue3-webpack-ts)
