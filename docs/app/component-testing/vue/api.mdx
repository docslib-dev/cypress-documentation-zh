# Vue API

## 方法

### mount

```js
import { mount } from 'cypress/vue'
```

<table class="api-table table-list">
  <tr>
    <td>描述</td>
    <td>
      用于独立挂载Vue组件。负责在Cypress的沙盒iframe中渲染组件，并处理任何框架特定的清理工作。
    </td>
  </tr>
  <tr>
    <td>签名</td>
    <td>
      mount(originalComponent: &lbrace; new (...args: any[]): {}; __vccOpts:
      any; &rbrace;, options?: MountOptions):
      Cypress.Chainable&lt;MountReturn&gt;
    </td>
  </tr>
</table>

<table class="api-table">
  <caption>mount参数</caption>
  <thead>
    <td>名称</td>
    <td>类型</td>
    <td>描述</td>
  </thead>
  <tr>
    <td>originalComponent</td>
    <td>new (...args: any[])</td>
    <td>要在测试中挂载的组件</td>
  </tr>
  <tr>
    <td>options</td>
    <td>MountOptions (可选)</td>
    <td>挂载组件的选项</td>
  </tr>
</table>

## 接口

### MountOptions

([Vue 3 MountingOptions](https://test-utils.vuejs.org/api/#mount) 或
[Vue Test Utils](https://test-utils.vuejs.org/))
