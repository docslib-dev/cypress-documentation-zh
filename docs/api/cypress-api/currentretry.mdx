# Cypress.currentRetry

`Cypress.currentRetry` 是一个表示当前
[测试重试](/app/guides/test-retries) 次数的数字。

:::caution

请注意 `Cypress.currentRetry` 只能在测试和
[测试钩子](/app/core-concepts/writing-and-organizing-tests#Hooks) 内部使用，
在测试和测试钩子外部使用时其值为 `null`。

:::

## 语法

```javascript
Cypress.currentRetry
```

## 示例

### 获取当前测试重试次数

```javascript
it('example', () => {
  expect(Cypress.currentRetry).to.eq(0)
})
```
