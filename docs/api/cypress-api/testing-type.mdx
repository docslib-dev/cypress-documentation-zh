# Cypress.testingType

`Cypress.testingType` 返回当前测试类型，该类型由您在 Cypress Launchpad 中的选择决定。`Cypress.testingType` 会返回 `e2e` 表示
[端到端测试](/app/core-concepts/testing-types#What-is-E2E-Testing) 或 `component` 表示
[组件测试](/app/core-concepts/testing-types#What-is-Component-Testing)。

## 语法

```javascript
Cypress.testingType // 返回 'e2e' 或 'component'
```

## 示例

### 测试类型

```javascript
it('正在运行组件测试模式', () => {
  expect(Cypress.testingType).to.equal('component')
})
```

### 条件判断

```javascript
it('执行不同的操作', () => {
  if (Cypress.testingType === 'e2e') {
    cy.exec('something')
  } else {
    cy.exec('something else')
  }
})
```
