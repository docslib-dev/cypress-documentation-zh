# Cypress.version

`Cypress.version` 返回当前运行的 Cypress 版本。

## 语法

```javascript
Cypress.version // '1.1.0'
```

## 示例

### 条件判断

```javascript
const semver = require('semver')

if (semver.gte(Cypress.version, '1.1.3')) {
  it('has Cypress.platform', () => {
    expect(Cypress.platform).to.be.a('string')
  })
}
```

:::note

**提示：** 你可以使用 [semver](https://github.com/npm/node-semver#readme) 库来处理语义化版本。

:::
