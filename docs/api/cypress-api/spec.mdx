# Cypress.spec

`Cypress.spec` 返回当前测试规范的属性。

## 语法

```javascript
Cypress.spec // 返回规范对象
```

## 示例

### 输出规范信息

#### `Cypress.spec` 返回一个对象

```js
it('输出规范信息', () => {
  console.log(Cypress.spec)
  // {
  //   name: 'filter.cy.js',
  //   relative: 'cypress/e2e/filter.cy.js',
  //   absolute: '/Users/janelane/Dev/web-app/cypress/e2e/filter.cy.js',
  // }
})
```
