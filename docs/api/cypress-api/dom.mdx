# Cypress.dom

`Cypress.dom.method()` 是 Cypress 中一系列与 DOM 相关的辅助方法集合。

:::caution

实际上 `Cypress.dom` 还包含数十个未在下方文档中列出的方法。这些方法被 Cypress 内部几乎所有的内置命令所使用。

建议
[查阅此处的源代码](https://github.com/cypress-io/cypress/blob/develop/packages/driver/src/dom/index.ts)
以了解所有方法及其功能。

:::

## 语法

```javascript
Cypress.dom.isHidden(element)
```

## 示例

### 是否已附加

**返回布尔值，表示元素是否已附加到 DOM 中。**

```javascript
cy.get('button').then(($el) => {
  Cypress.dom.isAttached($el) // true
})
```

### 是否为后代元素

**返回布尔值，表示一个元素是否是另一个元素的后代。**

```javascript
cy.get('div').then(($el) => {
  Cypress.dom.isDescendent($el.parent(), $el) // true
})
```

### 是否已分离

**返回布尔值，表示元素是否已从 DOM 中分离。**

```javascript
cy.get('button').then(($el) => {
  Cypress.dom.isDetached($el) // false
})
```

### 是否为文档类型

**返回布尔值，表示节点是否为 document 类型。**

```javascript
cy.get('p').then(($el) => {
  Cypress.dom.isDocument($el) // false
})
```

### 是否为 DOM 对象

**返回布尔值，表示对象是否为 DOM 对象。**

```javascript
cy.get('body').then(($el) => {
  Cypress.dom.isDom($el) // true
})
```

### 是否为 DOM 元素

**返回布尔值，表示对象是否为 DOM 元素。**

```javascript
cy.get('p').then(($el) => {
  Cypress.dom.isElement($el) // true
})
```

### 是否可聚焦

**返回布尔值，表示元素是否可以接收焦点。**

Cypress 内部广泛使用此方法来判断元素是否隐藏，
[主要用于可操作性检查](/app/core-concepts/interacting-with-elements)。

```javascript
cy.get('input').then(($el) => {
  Cypress.dom.isFocusable($el) // true
})
```

### 是否已聚焦

**返回布尔值，表示元素当前是否拥有焦点。**

```javascript
cy.get('button').then(($el) => {
  Cypress.dom.isFocused($el)
})
```

### 是否隐藏

**返回布尔值，表示元素是否隐藏。**

Cypress 内部广泛使用此方法来判断元素是否隐藏，
[主要用于可操作性检查](/app/core-concepts/interacting-with-elements)。

```javascript
cy.get('p').then(($el) => {
  Cypress.dom.isHidden($el) // false
})
```

### 是否为 jQuery 对象

**返回布尔值，表示对象是否为 jQuery 对象。**

```javascript
cy.get('input').then(($el) => {
  Cypress.dom.isJquery($el)
})
```

### 是否可滚动

**返回布尔值，表示元素是否可滚动。**

Cypress 内部广泛使用此方法来判断元素是否可滚动，
[主要用于可操作性检查](/app/core-concepts/interacting-with-elements)。

```javascript
cy.get('body').then(($el) => {
  Cypress.dom.isScrollable($el) // true
})
```

### 是否可见

**返回布尔值，表示元素是否可见。**

```javascript
cy.get('img').then(($el) => {
  Cypress.dom.isVisible($el) // true
})
```

### 是否为 window 对象

**返回布尔值，表示对象是否为 window 对象。**

```javascript
cy.get(window).then(($el) => {
  Cypress.dom.isWindow($el) // true
})
```

### 解包

**从 jQuery 对象中提取原始元素组成的数组。**

```javascript
cy.get('body').then(($el) => {
  Cypress.dom.unwrap($el)
})
```

### 包装

**通过将对象包装在 jQuery 中返回 jQuery 对象。**

```javascript
cy.get('p').then(($el) => {
  Cypress.dom.wrap($el)
})
```
