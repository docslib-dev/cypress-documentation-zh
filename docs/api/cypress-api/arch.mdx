# Cypress.arch

`Cypress.arch` 返回底层操作系统的 CPU 架构名称，该名称来自 Node 的 [`os.arch()`](https://nodejs.org/api/os.html#os_os_arch)。

尽管 Cypress 在浏览器中运行，但它会自动提供此属性以便在测试中使用。

## 语法

```javascript
Cypress.arch // 'x64'
```

## 示例

### CPU 架构

```javascript
it('has expected CPU architecture', () => {
  expect(Cypress.arch).to.be.oneOf(['x64', 'ia32'])
})
```

### 条件判断

```javascript
it('does something differently', () => {
  if (Cypress.arch === 'x64') {
    cy.exec('something')
  } else {
    cy.exec('something else')
  }
})
```
