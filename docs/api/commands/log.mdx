# log

在 Cypress 命令日志中打印消息。

## 语法

```javascript
cy.log(message)
cy.log(message, args...)
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.log('created new user')
```

### 参数

<Icon name="angle-right" /> **message _(String)_**

要打印到 Cypress 命令日志中的消息。支持 Markdown 格式的消息。

<Icon name="angle-right" /> **args...**

要打印到 Cypress 命令日志中的额外参数。参数数量没有限制。

<HeaderYields />

- `cy.log()` 返回 `null`。

## 示例

### 消息

#### 在命令日志中打印消息。

```javascript
cy.click('Login')
cy.log('Login successful')
```

### 参数

#### 在命令日志中打印带参数的消息。

```javascript
cy.log('events triggered', events)
```

## 规则

<HeaderRequirements />

- `cy.log()` 必须通过 `cy` 链式调用。

<HeaderAssertions />

- `cy.log()` 不能链式调用任何断言。

<HeaderTimeouts />

- `cy.log()` 不会超时。

## 命令日志

**_在命令日志中打印带参数的消息。_**

```javascript
cy.log('log out any message we want here')
cy.log('another message', ['one', 'two', 'three'])
```
