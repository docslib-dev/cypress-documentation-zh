# clearAllCookies

清除所有浏览器cookie。

:::caution

Cypress 会在每次测试_之前_自动清除所有cookie，以防止在启用[测试隔离](/app/core-concepts/writing-and-organizing-tests#Test-Isolation)时状态在测试之间共享。除非你需要在一个测试内部清除所有cookie或测试隔离被禁用，否则不应使用此命令。

:::

## 语法

```javascript
cy.clearAllCookies()
cy.clearAllCookies(options)
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.clearAllCookies() // 清除所有cookie
```

### 参数

<Icon name="angle-right" /> **options _(对象)_**

传入一个选项对象来改变`cy.clearAllCookies()`的默认行为。

| 选项      | 默认值                                                     | 描述                                                                         |
| --------- | ----------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| `log`     | `true`                                                      | 在[命令日志](/app/core-concepts/open-mode#Command-Log)中显示该命令 |
| `timeout` | [`responseTimeout`](/app/references/configuration#Timeouts) | 等待`cy.clearAllCookies()`解析的时长，超过则[超时](#Timeouts)   |

<HeaderYields />

- `cy.clearAllCookies()` 返回 `null`。

## 规则

<HeaderRequirements />

- `cy.clearAllCookies()` 需要链式调用 `cy`。

<HeaderAssertions />

- `cy.clearAllCookies()` 不能链式调用任何断言。

<HeaderTimeouts />

- `cy.clearAllCookies()` 不应超时。

:::caution

由于 `cy.clearAllCookies()` 是异步的，理论上在与Cypress内部自动化API通信时可能会出现超时。但实际上这种情况几乎不会发生。

:::
