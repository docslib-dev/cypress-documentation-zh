# title

获取当前活动页面的 `document.title` 属性。

## 语法

```javascript
cy.title()
cy.title(options)
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.title() // 以字符串形式返回文档标题
```

### 参数

<Icon name="angle-right" /> **options _(Object)_**

传入一个选项对象来改变 `cy.title()` 的默认行为。

| 选项      | 默认值                                                           | 描述                                                                         |
| --------- | ----------------------------------------------------------------- | --------------------------------------------------------------------------- |
| `log`     | `true`                                                            | 在[命令日志](/app/core-concepts/open-mode#Command-Log)中显示该命令           |
| `timeout` | [`defaultCommandTimeout`](/app/references/configuration#Timeouts) | 在[超时](#Timeouts)前等待 `cy.title()` 解析的时间                           |

<HeaderYields />

- `cy.title()` 返回当前页面的 `document.title` 属性。
- `cy.title()` 是一个查询命令，可以安全地链式调用其他命令。

## 示例

### 无参数

#### 断言文档标题为 "My Awesome Application"

```javascript
cy.title().should('eq', 'My Awesome Application')
```

## 规则

<HeaderRequirements />

- `cy.title()` 必须通过 `cy` 链式调用。

<HeaderAssertions />

- `cy.title()` 会自动[重试](/app/core-concepts/retry-ability)，直到所有链式断言通过。

<HeaderTimeouts />

- `cy.title()` 可能会因等待添加的断言通过而超时。
