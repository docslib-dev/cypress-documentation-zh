# end

结束命令链。

## 语法

```javascript
.end()
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.contains('ul').end() // 返回'null'而不是'ul'元素
```

<Icon name="exclamation-triangle" color="red" /> **错误用法**

```javascript
cy.end()
```

<HeaderYields />

- `.end()` 返回 `null`。

## 示例

`.end()` 在您想结束命令链并强制下一条命令不接收上一条命令的返回结果时非常有用。

```javascript
cy.contains('用户: Cheryl')
  .click()
  .end() // 返回null
  .contains('用户: Charles')
  .click() // contains现在会在文档中查找内容
```

或者，您也可以直接从 `cy` 开始一个新的命令链。

```javascript
cy.contains('用户: Cheryl').click()
cy.contains('用户: Charles').click() // contains现在会在文档中查找内容
```

## 规则

<HeaderRequirements />

- `.end()` 必须在上一条命令后链式调用。

<HeaderAssertions />

- `.end()` 不能链式调用任何断言。

<HeaderTimeouts />

- `.end()` 不会超时。

## 命令日志

- `.end()` 不会在命令日志中记录
