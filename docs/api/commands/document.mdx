# document

获取当前活动页面的 `window.document` 对象。

## 语法

```javascript
cy.document()
cy.document(options)
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.document() // 返回 window.document 对象
```

### 参数

<Icon name="angle-right" /> **options _(Object)_**

传入一个选项对象来改变 `cy.document()` 的默认行为。

| 选项      | 默认值                                                           | 描述                                                                         |
| --------- | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| `log`     | `true`                                                            | 在[命令日志](/app/core-concepts/open-mode#Command-Log)中显示该命令 |
| `timeout` | [`defaultCommandTimeout`](/app/references/configuration#Timeouts) | 等待 `cy.document()` 解析的超时时间，超过则[超时](#Timeouts)          |

<HeaderYields />

- `cy.document()` 返回 `window.document` 对象。
- `cy.document()` 是一个查询命令，可以安全地链式调用其他命令。

## 示例

### 无参数

#### 获取 document 并执行操作

```javascript
cy.document().then((doc) => {
  // 操作 document 元素
})
```

#### 对 document 进行断言

```javascript
cy.document().its('contentType').should('eq', 'text/html')
```

## 规则

<HeaderRequirements />

- `cy.document()` 必须通过 `cy` 链式调用。

<HeaderAssertions />

- `cy.document()` 会自动
  [重试](/app/core-concepts/retry-ability) 直到所有链式断言通过

<HeaderTimeouts />

- `cy.document()` 可能会因等待添加的断言通过而超时。
