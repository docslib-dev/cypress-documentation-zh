# clearAllSessionStorage

清除测试交互过的所有源的
[`sessionStorage`](https://developer.mozilla.org/en-US/docs/Web/API/Window/sessionStorage)
数据。

:::caution

Cypress 会在每次测试_前_自动清除所有sessionStorage，以防止当
[测试隔离](/app/core-concepts/writing-and-organizing-tests#Test-Isolation)
启用时状态在测试之间共享。除非你需要在单个测试内清除sessionStorage或测试隔离被禁用，否则不应使用此命令。

:::

## 语法

```javascript
cy.clearAllSessionStorage()
cy.clearAllSessionStorage(options)
```

### 用法

<Icon name="check-circle" color="green" /> **正确用法**

```javascript
cy.clearAllSessionStorage()
```

### 参数

<Icon name="angle-right" /> **options _(Object)_**

传入一个选项对象来改变`cy.clearAllSessionStorage()`的默认行为。

| 选项 | 默认值 | 描述                                                                         |
| ------ | ------- | ----------------------------------------------------------------------------------- |
| `log`  | `true`  | 在[命令日志](/app/core-concepts/open-mode#Command-Log)中显示该命令 |

<HeaderYields />

- `cy.clearAllSessionStorage()` 返回 `null`。

## 规则

<HeaderRequirements />

- `cy.clearAllSessionStorage()` 需要链式调用 `cy`。

<HeaderAssertions />

`cy.clearAllSessionStorage()` 不能链式调用任何断言。

<HeaderTimeouts />

- `cy.clearAllSessionStorage()` 不会超时。
